SC
Semantic Chunking
Documents split by cosine similarity drops between adjacent sentences. Preserves semantic coherence unlike fixed-size splitting.
|
QD
Query Decomposition
Multi-hop questions broken into targeted sub-queries using rule-based patterns to maximise retrieval coverage.
↓
FR
FAISS Retrieval
Dense vector search using BAAI/bge-small-en (384-dim). Top-20 candidates retrieved per sub-query from the FAISS index.
↓
CR
Cross-Encoder Reranking
BAAI/bge-reranker-base scores each candidate jointly with the query. Far more accurate than bi-encoder similarity alone.
↓
WA
Weighted Evidence Aggregation
EvidenceScore = 0.5×similarity + 0.4×reranker + 0.1×density — composite scoring filters noise and ranks evidence by overall relevance.
↓
LM
LLM Generation — Mistral 7B
Top evidence chunks assembled into a context window. Mistral 7B generates the final grounded answer via Ollama (localhost:11434).
